Etap 3: Implementacja analizatora składniowego

Program częściowo implementuje translator XML do JSON. Program napisany z użyciem środowiska Visual C++ 2012.
Program można skompilować, lub odrazu uruchomić (podkatalog Degug/xml2js.exe)

Program udostępnia dwa tryby działania:
1. Z interfejsem linii polecen
   Dla odpalenia trybu trzeba uruchomic program w podkatalogu xml2js/Degug/xml2js.exe
   
2. Tryb szybki: jest uruchamiany gdy przekazany jest argument z nazwą pliku wejściowego XML. 
   W takim trybie program odrazu generuje plik wyjsciowy JSON.
   
 
TESTOWANIE
Na danym etapie juz został napisany skrypt pythonowy dla testowania zautomatyzowanego programu. 
Dla urochomienia skryptu((podkatalog xml2js/Degug/) trzeba uruchomić jeden z plików runTests. w załeżności od systemu operacyjnego i upodobań użytkownika.
Dla uruchomienia skryptu na komputerze powinien być zainstalowany Python.

Bardziej szczegółowy opis znajdzie się w dokumentacji końcowej do projektu.
